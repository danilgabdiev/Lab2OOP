Ошибки памяти

Ошибка: Выход за пределы массива.
Исправление: Проверьте индексы и используйте std::vector.
int arr[10];
arr[10] = 5; // Ошибка: индекс выходит за границу
Ошибка: Утечка памяти.
Исправление: Освобождайте память с помощью delete[] или используйте умные указатели.
int* p = new int[10];
// Не вызван delete[]
delete[] p;
Ошибка: Двойное освобождение памяти.
Исправление: После delete устанавливайте указатель в nullptr.
int* p = new int;
delete p;
p = nullptr;
Ошибка: Переполнение буфера.
Исправление: Используйте безопасные функции для копирования строк или std::string.
char buf[10];
strcpy(buf, "слишком длинная строка");
Ошибка: Висячий указатель.
Исправление: Не обращайтесь к указателю после delete.
int* p = new int(5);
delete p;
// Ошибка: обращение к уже освобождённой памяти

Ошибки указателей

Ошибка: Разыменование nullptr.
Исправление: Проверяйте, что указатель не равен nullptr.
int* p = nullptr;
if(p) { *p = 10; }
Ошибка: Использование после освобождения памяти.
Исправление: После delete устанавливайте указатель в nullptr.
int* p = new int;
delete p;
p = nullptr;
Ошибка: Неинициализированный указатель.
Исправление: Инициализируйте указатель, например, значением nullptr.
int* p = nullptr;
Ошибка: Неправильная арифметика указателей.
Исправление: Следуйте правилам арифметики указателей и проверяйте границы массива.
int arr[5];
int* p = arr;
p = p + 5; // может выйти за границы
Ошибка: Повреждение памяти при копировании указателей.
Исправление: Используйте безопасное копирование или контейнеры STL.
int* p = new int[5];
// Некорректное копирование
memcpy(p, p+1, 5*sizeof(int));

Арифметические ошибки

Ошибка: Переполнение числового типа.
Исправление: Проверяйте диапазоны или используйте тип int64_t.
unsigned int x = 4294967295;
x = x + 1;
Ошибка: Деление на ноль.
Исправление: Проверьте делитель перед операцией деления.
int a = 10, b = 0;
if(b != 0) {
  int c = a / b;
}
Ошибка: Потеря точности при вычислениях.
Исправление: Используйте тип double и методы округления.
double a = 0.1, b = 0.2;
double c = a + b;
Ошибка: Underflow (недополнение).
Исправление: Проверьте нижние границы перед операцией.
unsigned int x = 0;
x--;
Ошибка: Ошибка округления.
Исправление: Используйте корректные методы округления.
double a = 2.675;
double r = Math.round(a * 100) / 100.0;
